EVOLUÇÃO DO PLUGIN AMAZON POPUPS - VERSÃO 3.0

🎯 HISTÓRICO DE MELHORIAS

=== VERSÃO 2.8 - CORREÇÃO DE NAVEGAÇÃO ===
- Problemas com controles de carrossel
- IDs conflitantes entre desktop e mobile
- Navegação não funcionava corretamente

=== VERSÃO 2.9 - CORREÇÃO BOOTSTRAP ===
- Bootstrap não carregava em widgets
- Dependências problemáticas
- Carregamento condicional melhorado

=== VERSÃO 3.0 - REMOÇÃO DO CARROSSEL ===
✅ DECISÃO TÉCNICA: Carrossel Bootstrap REMOVIDO
✅ SUBSTITUIÇÃO: Sistema de cards estáveis
✅ RESULTADO: Plugin mais confiável

=== FUNCIONALIDADES ATUAIS ===

🎨 TEMPLATES DISPONÍVEIS:
• [amazon_products template="cards"] - Cards informativos
• [amazon_products template="grid"] - Grid simples  
• [amazon_products template="list"] - Lista vertical

🔍 SHORTCODE COMPLETO:
• [amazon_product_details product_id="123"] - Informações detalhadas

📋 FILTROS POR CATEGORIA:
• [amazon_products category="tecnologia"] - Por categoria
• [amazon_products post_type="product" category="eletronicos"] - WooCommerce
